Origami for Sublime Text

Panes

CMD+K Arrow

Travel focus to adjacent pane

CMD+K SHIFT-Arrow

Carry current file to adjacent pane

CMD+K OPTION-Arrow

Clone current file to adjacent pane

CMD+K CMD-Arrow

Create an adjacent pane

CMD+K CMD-SHIFT-Arrow

Destroy an adjacent pane

Zoom / Unzoom

CMD+K CMD+Z

Zoom on current pane

CMD+K CMD+SHIFT+Z

Unzoom and equally space all panes
