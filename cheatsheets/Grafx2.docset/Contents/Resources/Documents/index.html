Grafx2

Move

Arrow Up

Scroll up

Scrolls the picture up, both in magnify and normal mode. This hotkey cannot be removed.

Arrow Down

Scroll down

Scrolls the picture down, both in magnify and normal mode. This hotkey cannot be removed.

Arrow Left

Scroll left

Scrolls the picture to the left, both in magnify and normal mode. This hotkey cannot be removed.

Arrow Right

Scroll right

Scrolls the picture to the right, both in magnify and normal mode. This hotkey cannot be removed.

SHIFT+Arrow Up

Faster scroll up

Used to scroll upwards in the picture fast, either in magnify and normal mode.

SHIFT+Arrow Down

Faster scroll down

Used to scroll downwards in the picture fast, either in magnify and normal mode.

SHIFT+Arrow Left

Faster scroll left

Used to scroll to the left in the picture fast, either in magnify and normal mode.

SHIFT+Arrow Right

Faster scroll right

Used to scroll to the right in the picture fast, either in magnify and normal mode.

OPTION+Arrow Up

Slower scroll up

Used to scroll upwards in the picture pixel by pixel, either in magnify and normal mode.

OPTION+Arrow Down

Slower scroll down

Used to scroll downwards in the picture pixel by pixel, either in magnify andnormal mode. normal mode.

OPTION+Arrow Left

Slower scroll left

Used to scroll to the left in the picture pixel by pixel, either in magnify and normal mode.

OPTION+Arrow Right

Slower scroll right

Used to scroll to the right in the picture pixel by pixel, either in magnify and normal mode.

Mouse emulation

CTRL+Arrow Up

Move mouse cursor 1 pixel up

Used to simulate a very small mouse deplacement upwards. Its very useful when you want a ultra-high precision.

CTRL+Arrow Down

Move mouse cursor 1 pixel down

Used to simulate a very small mouse deplacement downwards. Its very useful when you want a ultra-high precision.

CTRL+Arrow Left

Move mouse cursor 1 pixel left

Used to simulate a very small mouse deplacement to the left. Its very useful when you want a ultra-high precision.

CTRL+Arrow Right

Move mouse cursor 1 pixel right

Used to simulate a very small mouse deplacement to the right. Its very useful when you want a ultra-high precision.

Space

Simulate left mouse click

Used to simulate a click with the left mouse button.. Its very useful when you want a ultra-high precision.

SHIFT+Space

Simulate right mouse click

Used to simulate a click with the right mouse button.. Its very useful when you want a ultra-high precision.

Settings

SHIFT+F10

Settings

Opens a menu which permits you to set the dimension of your picture, and to modify some parameters of the program.

Return

Screen resolution menu

Opens a menu where you can choose the dimensions of the screen in which you want to draw among the numerous X and SVGA proposed modes.

SHIFT+Return

"Safety" resolution

Set resolution to 320x200. This can be useful if you choosed a resolution that is not supported by your monitor and video card. Cannot be removed.

F10

Show/hide option menu

Switch the tool bar display on/off. This hot-key cannot be removed.

F9

Show/hide cursor

Switch the cursor display on/off. This only works on the "small cross" and "hand" cursors.

Brush

Del

Set paintbrush to 1 pixel

Useful when you want to use a "single-pixel-brush".

F4

Paintbrush choice

Opens a menu where you can choose a paintbrush out of 24 predefined ones.

SHIFT+F4

Monochrome brush

Turn your current user-defined brush into a single colored one. All non-transparent colors are set to current foreground color.

Paint

U

Undo (Oops!)

Cancel the last action which modified the picture. This has no effect after a jump to the spare page, loading a picture or modifying its size.

SHIFT+U

Redo

Redo the last undone action. This has no effect after a jump to the spare page, loading a picture or modifying its size.

D

Freehand drawing

Set the drawing mode to the classical freehand one.

SHIFT+D

Switch freehand drawing mode

Switch between the 3 ways to use freehand drawing. These modes are: continuous, discontinuous and point by point.

CTRL+D

Continuous freehand drawing

Switch directly to continuous freehand drawing mode.

L

Line

Allows you to draw lines.

SHIFT+L

Knotted lines

Allows you to draw linked lines. This mode can also be called "Polyline".

A

Spray

Allows you to spray brushes randomly in the picture.

SHIFT+A

Spray menu

Opens a menu in which you can configure the spray flow and size.

F

Flood-fill

Allows you to fill an area of the picture made of pixels of the same color.

SHIFT+F

Replace color

Allows you to replace all the pixels of the color pointed by the mouse with the fore-color or the back-color.

I

Beziers curves

Allows you to draw Beziers curves.

SHIFT+I

Beziers curve with 3 or 4 points

Allows you to choose whether you want to draw Beziers curves with 3 or 4 points.

R

Empty rectangle

Allows you to draw a rectangle using the brush.

SHIFT+R

Filled rectangle

Allows you to draw a filled rectangle.

C

Empty circle

Allows you to draw a circle using the brush.

CTRL+C

Empty ellipse

Allows you to draw an ellipse using the brush.

SHIFT+C

Filled circle

Allows you to draw a filled circle.

CTRL+SHIFT+C

Filled ellipse

Allows you to draw a filled ellipse.

N

Empty polygon

Allows you to draw a polygon using the brush.

CTRL+N

Empty "polyform"

Allows you to draw a freehand polygon using the brush.

SHIFT+N

Filled polygon

Allows you to draw a filled polygon.

CTRL+SHIFT+N

Filled "polyform"

Allows you to draw a filled freehand polygon.

OPTION+R

Rectangle with gradation

Allows you to draw a rectangle with a color gradation.

OPTION+G

Gradation menu

Allows you to configure the way color gradations are calculated.

OPTION+C

Sphere with gradation

Allows you to draw a rectangle with a color gradation.

OPTION+SHIFT+C

Ellipse with gradation

Allows you to draw an ellipse filled with a color gradation.

T

Text

Opens a menu which permits you to type in a character string and to choose a font, and then creates a new user-defined brush fitting to your choices.

,

Shrink paintbrush

Decrease the width of the paintbrush if it is special circle or square.

.

Enlarge paintbrush

Increase the width of the paintbrush if it is special circle or square.

Picture

KP 5

Adjust picture

Allows you to move the whole picture in order to re-center it. Notice that what gets out from a side reappears on the other.

SHIFT+KP 5

Flip/shrink picture menu

Opens a menu which allows you to flip the picture horizontally/vertically or to shrink it to half-scale horizontally and/or vertically.

Effects

E

Drawing effects

Opens a menu where you can enable/disable and configure the drawing effects listed below.

F5

Shade mode

Allows you to shade or lighten some pixels of the picture belonging to a color range, in addition of any drawing tool.

SHIFT+F5

Shade menu

Opens a menu where you can choose color ranges to use with the Shade mode. This menu also contains parameters used both in Shade and Quick-shade modes.

CTRL+F5

Quick-shade mode

Does the same thing as shade mode with a simpler method (faster to define but a bit less powerful).

CTRL+SHIFT+F5

Quick-shade menu

Opens a menu where you can define the parameters of the quick-shade mode.

F6

Stencil mode

Allows you to mask colors that must not be affected when you are drawing.

SHIFT+F6

Stencil menu

Opens a menu where you can choose colors masked by the Stencil mode.

OPTION+F6

Mask mode

Allows you to mask colors of the spare page that will keep you from drawing. This mode should be called "True stencil".

OPTION+SHIFT+F6

Mask menu

Opens a menu where you can choose colors for the Mask mode.

G

Grid mode

Force the cursor to snap up grid points.

SHIFT+G

Grid menu

Open a menu where you can configure the grid used by Grid mode.

CTRL+G

Sieve mode

Only draws pixels on certain positions matching with a sieve.

CTRL+SHIFT+G

Sieve menu

Opens a menu where you can configure the sieve.

CTRL+OPTION+G

Invert sieve

Inverts the pattern defined in the Sieve menu.

F7

Colorize mode

Allows you to colorize the pixels on which your brush is pasted. This permits you to make transparency effects.

SHIFT+F7

Colorize menu

Opens a menu where you can give the opacity percentage for Colorize mode.

F8

Smooth mode

Soften pixels on which your brush is pasted.

SHIFT+F8

Smooth menu

Opens a menu where you can define the Smooth matrix.

OPTION+F8

Smear mode

Smears the pixels when you move your brush on the picture.

OPTION+B

Tiling mode

Puts parts of the brush where you draw.

OPTION+SHIFT+B

Tiling menu

Opens a menu where you can configure the origin of the tiling.

Brush

B

Classical brush grabbing

Allows you to pick a brush defined within a rectangle.

CTRL+B

"Lasso" brush grabbing

Allows you to pick a brush defined within a freehand polygon.

SHIFT+B

Get previous brush back

Restore the last user-defined brush.

X

Horizontal brush flipping

Reverse brush horizontally.

Y

Vertical brush flipping

Reverse brush vertically.

Z

90ø brush rotation

Rotate the user-defined brush by 90ø (counter-clockwise).

SHIFT+Z

180ø brush rotation

Rotate the user-defined brush by 180ø.

S

Strech brush

Allows you to resize the user-defined brush.

SHIFT+S

Distort brush

Allows you to distort the user-defined brush.

O

Outline brush

Outlines the user-defined brush with the fore color.

SHIFT+O

Nibble brush

Deletes the borders of the user-defined brush. This does the opposite of the Outline option.

F11

Get colors from brush

Copy colors of the spare page that are used in the brush.

F12

Recolorize brush

Recolorize pixels of the user-defined brush in order to get a brush which looks like the one grabbed in the spare page.

W

Rotate by any angle

Rotate the brush by an angle that you can define.

CTRL+OPTION+B

Brush effects menu

Opens a menu which proposes different effects on the user-defined brush.

CTRL+KP 5

Center brush attachment point

Set the attachement of the user-defined brush to its center.

CTRL+Home

Top-left brush attachment point

Set the attachement of the user-defined brush to its top-left corner.

CTRL+Page Up

Top-right brush attachment point

Set the attachement of the user-defined brush to its top-right corner.

CTRL+End

Bottom-left brush attachment point

Set the attachement of the user-defined brush to its bottom-left corner.

CTRL+Page Down

Bottom-right brush attachment point

Set the attachement of the user-defined brush to its bottom-right corner.

Colors

`

Pipette

Allows you to copy the color of a pixel in the picture into the foreground or background color.

SHIFT+`

Swap foreground/background colors

Invert foreground and background colors.

P

Palette menu

Opens a menu which allows you to modify the current palette.

SHIFT+P

Secondary palette menu

Opens a menu which allows you to define color series and some tagged colors.

CTRL+P

Exclude colors menu

Opens a menu which allows you to define the colors you dont want to use in modes such as Smooth and Transparency, or when remapping a brush.

Page Up

Scroll palette to the left

Scroll palette in the tool bar to the left, column by column.

Page Down

Scroll palette to the right

Scroll palette in the tool bar to the right, column by column.

SHIFT+Page Up

Scroll palette to the left faster

Scroll palette in the tool bar to the left, 8 columns by 8 columns.

SHIFT+Page Down

Scroll palette to the right faster

Scroll palette in the tool bar to the right, 8 columns by 8 columns.

]

Next foreground color

Set the foreground color to the next in the palette.

[

Previous foreground color

Set the foreground color to the previous in the palette.

SHIFT+]

Next background color

Set the background color to the next in the palette.

SHIFT+[

Previous background color

Set the background color to the previous in the palette.

=

Next user-defined forecolor

Set the foreground color to the next in the user-defined color series.

-

Previous user-defined forecolor

Set the foreground color to the previous in the user-defined color series.

SHIFT+=

Next user-defined backcolor

Set the background color to the next in the user-defined color series.

SHIFT+-

Previous user-defined backcolor

Set the background color to the previous in the user-defined color series.

Zoom

M

Magnifier mode

Allows you to zoom into the picture.

SHIFT+M

Zoom factor menu

Opens a menu where you can choose a magnifying factor.

KP +

Zoom in

Increase magnifying factor.

KP -

Zoom out

Decrease magnifying factor.

Divers

Q

Quit program

Allows you to leave the program. If modifications were not saved, confirmation is asked.

F1

Help and credits

Opens a window where you can get information about the program.

SHIFT+F1

Statistics

Displays miscellaneous more or less useful information.

Save/Load

F2

Save picture as...

Opens a file-selector that allows you to save your picture with a new path-name.

SHIFT+F2

Save picture

Saves your picture with the last name you gave it.

F3

Load picture

Opens a file-selector that allows you to load a new picture.

SHIFT+F3

Re-load picture

Re-load the current picture. This allows you to cancel modifications made since last saving.

CTRL+F2

Save brush

Opens a file-selector that allows you to save your current user-defined brush.

CTRL+F3

Load brush

Opens a file-selector that allows you to load a brush.

Pages

Tab

Jump to spare page

Swap current page and spare page.

SHIFT+Tab

Copy current page to spare page

Copy current page to spare page.

SHIFT+Del

Kill

Kills the current page. It actually removes the current page from the list of "Undo" pages.

Backspace

Clear page

Clears the picture with the first color of the palette (usually black).

SHIFT+Backspace

Clear page with backcolor

Clears the picture with the backcolor.

Notes

These are the default shortcuts. You can modify the actual shortcuts for your config in the helpscreen inside the program.
