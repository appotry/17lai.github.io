HipChat

Keyboard Shortcuts

CMD+/

Help

CMD+F

Search room history

CMD+OPTION+Arrow Down

Next room or 1-1 chat

CMD+OPTION+Arrow Up

Previous room or 1-1 chat

CMD+<number>

Go to room #

The Lobby or room at the top is 1, next room is 2, and so on.

CMD+I

Invite someone to a room

CMD+SHIFT+N

Create new room

CMD+T

Join new room or 1-1 chat

CMD+SHIFT+T

Reopen closed chats

CMD+W

Close current room or 1-1 chat

Formatting

/code code-block

Message with syntax highlighting

Displays your message with code syntax highlighting.

/quote message

Quote formatting

Use special message formatting when you quote someone.

s/wrong word/right word

Fix typo

Fix your typos, if corrected within one minute.

/clear

Clear history

Clear chat history in the current room or 1-1 chat. History comes back when you reopen the room or chat.

/me message

About yourself

Chat about yourself in the third person.

#color-hex

Colors of hex-value

Show hexadecimal web colors right in HipChat. Example: #ffffff shows a white box next to #ffffff.

Status Update Commands

/available message

Status "available"

Set your status as available (green) and optionally include a message. Example: /available Working from home.

/away message

Status "away"

Set your status as away (yellow) and optionally include a message. Example: /away Out to lunch.

/dnd message

Status "Do Not Disturb"

Set your status as Do Not Disturb (red) and optionally include a message. Example: /dnd Finishing proposal.

Room Related Commands

/join room-name

Enter room

HipChat web app only: enter an existing HipChat room. Example: /join IT Team.

/part

Leave current room

/topic new-topic

Change topic

Change the topic of the room you are currently in. Example: /topic All things IT.

Notes

  • There may be further /-commands depending on your HipChat Integrations.
