Textastic

Keyboard shortcuts for Textastic, a robust iOS text editor.

General

CMD+OPTION+B

Toggle sidebar

CMD+OPTION+P

Toggle preview

CMD+CTRL+OPTION+P

Preview in Safari

CMD+OPTION+T

Toggle symbol list

Files

CMD+N

New file/folder

CMD+E

Edit file

CMD+F

Toggle find/replace

CMD+OPTION+Arrow Up

Upload file

CMD+OPTION+Arrow Down

Download file

CMD+OPTION+I

Toggle file properties

Code Editor

CMD+[

Shift left

CMD+]

Shift right

CMD+CTRL+E

Expand Emmet abbreviation

CMD+CTRL+A

Wrap with Emmet abbreviation

Tabs

CTRL+Tab

Select next tab

CTRL+SHIFT+Tab

Select previous tab

CMD+1

Show first tab

CMD+2

Show second tab

CMD+3

Show third tab

CMD+4

Show fourth tab

CMD+5

Show fifth tab

CMD+6

Show sixth tab

CMD+7

Show seventh tab

CMD+8

Show eighth tab

CMD+9

Show ninth tab

CMD+0

Show tenth tab
